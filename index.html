Capstone Project - Social Recipes

Social Recipes is a "Reddit for recipes" app where users can create their own unique variations on popular dishes and upvote or downvote other user variations to end the debate over whether grandma's meatballs really are the best once and for all. Built using Ruby on Rails on the backend and Vue.js on the frontend, the app allows user to sign up and log in, and create new recipe variations, edit or delete them. Different cuisines are filterable by various interactive points on a map built using the Mapbox API.
